如何进行调研与组织

发表于2023-10-02|更新于2024-01-01|学习
|阅读量:

调研组织的流程

初步了解概况 → 判断需不需要别人帮助 → 确立调研主题 → 确立调研对象 → 开展调研 → 分析 → 决策 → 形成政策 → 宣传 → 使用 → 反馈

什么时候需要调研？

  • 调研是为了获得相关主题的大量正确的信息。
  • 而获得大量正确的信息，通常是为了制定政策或者策略。
  • 制定政策或策略是为了解决问题，因此当谈及怎么做的时候，必须要有调研。

什么时候需要形成工作小组

  • 难度比较高（如：存在专业壁垒）。
  • 涉及面广，需要考虑方方面面（含涉及多个部门）。

注：抽调人手，明确牵头对象。

什么时候需要咨询有经验的人？

  • 当无法明确指出事物的性质及范围时，则需要咨询
    • 专家
    • 重点企业
    • 有做过类似事情的人
    • 发挥领导的“引导”作用

什么时候需要对事物进行定性定量定标准（确立调研主题）

  • 当事物涉及面较小时（如征集家书，解决具体产品问题等）
    • 定性：
      • 看到事物发展的历史（古代、近代、现代）
      • 设计、建设、运营、保障
    • 定量：确定范围
    • 定标准：实现内容和形式的统一，在内容上要求真实，在形式上采用群众喜闻乐见的方式。

对于涉及面较广的事物，需要了解到什么信息才能确立调研主题

  • 通过网络方式了解XXX的概况，包括XXX在社会/市场/群众中的需求问题；
  • 通过动员部门成员，初步了解群众对于XXX的看法或建议；
  • 以及了解不同层次（地点+人员）情况，以便调研时可以选择典型；
  • 对于重大的问题，需要请有经验的人，给予指导；
  • 结合上述信息，在座谈会上确定调研主题。

确定调研对象

  • 什么时候需要确定调研地点？
    • 各区域情况有不同（事物分布不同，密度不同）
  • 调研对象（人）有哪些？
    • 整体了解（发放问卷）：通过对群众发放问卷，告知其意义，全面把握大体群众对XXX的看法。
    • 重点了解（分阶级）：
      • 【人】收入高、中、低
      • 【物】彩礼收入高、中、低的村子/家庭
      • 【有特殊作用的人】懂因果，明现状的老人，有斗争、先进意识的年轻人

如何开展调研

  1. 宣传：利用互联网、APP、公众号（线上）、线下社区公示栏等，在人流量多的地方宣传，告知群众这个活动的目的，是为了XXX，获得群众的认同，给群众切实的物质奖励和精神奖励，为XX奠定基础。
  2. 整体：通过问卷全面把握。
  3. 重点：一对一访谈。
  4. 其他
    1. 实地蹲点调研。
    2. 体验式调研：转变身份，以第一视角体验群众。
    3. 利用公益组织和先进团体，借助他们和群众的密切关系，以点带面，实现素材的广泛收集，同时需要为他们提供明确行动的指导方针（标准），并给奖励。
    4. 借助网络平台，走好网络群众路线，必要时，可借助大数据分析进行筛选，抓出有效信息。
    5. 对于投稿或参与实际活动，可定期去公示和实名表彰激励更多的人参与，激发内生动力。

如何分析与决策

  • 梳理成重点问题清单，针对性解决。
  • 形成调研报告，上报给领导审核，指正。

如何加大政策的宣传

  • 加强多渠道宣传。
  • 不但要告诉人们干什么和为什么要干，而且要规定什么应该做，什么不应该做和应该如何做。

如何根据反馈调整

  • 畅通保障机制，进行反馈调整。
