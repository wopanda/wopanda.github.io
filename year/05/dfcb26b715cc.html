安装Hexo

  • 前提

    • 安装Git
    • 安装Node.js
      • 下载 | Node.js 中文网 (nodejs.cn)
      • 环境变量需要安装，参考视频 Node.js安装及环境变量配置_哔哩哔哩_bilibili
      • 版本要大于12.3，这个与视频不同

  • OB目标目录下，调用命名

    • 安装hexo npm install -g hexo-cli
    • 初始化：hexo init

使用 github.io 展示网页

  • 测试本地是否可用
    • hexo g
    • hexo s
    • hexo clean
    • 将网址复制到浏览器上打开
  • 发布到pages
    • 【静态博客】部署到 GitHub_哔哩哔哩_bilibili
    • npm install hexo-deployer-git --save
    • hexo d
